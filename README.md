#
### TESTED ON :
* Termux
* Parrot Linux OS
### REQUIREMENTS :
* internet
* Python urllib3
* Python json
* Python http

## INSTALLATION [Termux] :

* `apt-get update -y`
* `apt-get upgrade -y`
* `pkg install python -y`
* `pkg install python2 -y`
* `pkg install git -y`
* `pip install -r requirements.txt`
* `git clone https://github.com/AbdulConsole/web-header-crawler`
* `cd $HOME`
* `ls`
* `cd web-header-crawler`
* `ls`
* `python whc.py`
